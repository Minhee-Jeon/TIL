## 변수와 출력함수   
```python
a, b, c = 3, 2, 1  # 한번에 변수 선언과 대입 가능
print(a, b, c)  # 3 2 1   
   
# 값 교환
a, b = 1, 2
a, b = b, a  # 값 교환이 이루어짐   

# 변수 타입
a = 12.123456789123456789
print(a)  # 12.1234567891234567 실수는 8byte 이후에는 값손실
print(type(a))  # <class 'float'>

'student' # str

# 출력 방식   
print("number")  # number

a, b, c = 1, 2, 3
print(a,b,c)  # 1 2 3
print(a,b,c,sep=', ')  # 1, 2, 3
print(a,b,c,sep='')  # 123
```
